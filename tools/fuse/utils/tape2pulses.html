NAME

tape2pulses — Sinclair ZX Spectrum tape pulse dumper

SYNOPSIS

tape2pulses infile outfile

DESCRIPTION

tape2pulses dumps the pulse information from ZX Spectrum tape images to text files.

OPTIONS

−h, −−help

give brief usage help, listing available options.

−V, −−version

output version information.

infile

specifies the tape file to be converted. This file can be in any of the tape formats supported by libspectrum(3).

outfile

specifies the output text file, if set to “−” will write the file to standard output.

BUGS

None known.
